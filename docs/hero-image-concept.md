# MSPI Hero Image Concept

## WWDD Constraints Applied

1. **NO dashboards** - "Dashboards are ego boards"
2. **NO fake product screenshots** - form before function, easy before hard
3. **NO competitor comparisons** - we don't compare ourselves to any company or market
4. **Must be CONCEPTUAL** - represent the value proposition, not the product

## Core MSPI Concepts to Visualize

From WWDD: **"Intelligence is compression."**

From the site:
- "The intelligence layer your MSP deserves"
- Cultivation vs forensics: "Daily tending of the revenue garden"
- Acceleration, not constant velocity
- Flywheel: data → insights → models → automation → outcomes → data
- "Intelligence Multiplied"

## Rejected Approaches

| Approach | Why Rejected (WWDD) |
|----------|---------------------|
| Dashboard mockup | Ego board, fake product |
| UI screenshots | Form before function |
| Competitor comparison | We don't compare |
| Stock photos of people | Generic, not conceptual |
| Charts and graphs | Ego board territory |

## Selected Concept: Intelligence Compression

**Visual metaphor for "Intelligence is compression"**

An abstract visualization showing raw data transforming into refined intelligence:

- **Base layer**: Scattered particles, fragments, chaos (raw data)
- **Middle layer**: Streams coalescing, organizing, flowing upward
- **Top layer**: Crystalline form, concentrated light, refined intelligence

This represents the MSPI value proposition without showing a fake product.

## Image Generation Prompt

```
Abstract minimalist visualization of intelligence compression.

Base: scattered gray particles and data fragments dispersing outward.

Middle: particles organizing into flowing streams, converging upward,
becoming more ordered and luminous.

Top: streams coalesce into a glowing crystalline form representing
refined intelligence. Concentrated, pure, actionable.

Color palette:
- Base: muted grays, cool tones
- Transition: silver to teal gradient
- Peak: luminous teal (#06d6a0) glow

Style: Clean white background. Modern, professional, elegant.
No text. No UI elements. No dashboards. No people.
Pure abstract concept art.

Aspect ratio: 16:9 (hero banner)
Quality: High fidelity, suitable for web hero section
```

## Why This Works (WWDD Aligned)

1. **Conceptual, not literal** - represents the value prop, not fake UI
2. **No comparison** - unique visual language, not derivative
3. **Hard before easy** - the concept comes from the actual philosophy
4. **Observable truth** - visualizes what MSPI actually does (compress intelligence)
5. **Not an ego board** - doesn't show state, shows transformation
